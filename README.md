# Gramps Descendants Timeline

Une nouvelle vue chronologique de descendance pour Gramps.

Contrairement aux arbres généalogiques classiques,
DescendantsTimeline représente les descendants
sur une véritable échelle du temps.

Les mariages, remariages, générations et durées de vie
deviennent immédiatement lisibles.

Le projet est entièrement développé en Python
pour Gramps 6.0.x.

Fonctionnalités

✔ Timeline chronologique
✔ Descendance
✔ Mariages
✔ Remariages
✔ Zoom
✔ Export SVG
✔ Export PDF
✔ Export PNG
